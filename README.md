APTime helps APU students find the best time to hang out, eat together, or plan group activities by comparing timetables across different intakes. Simply enter your intake codes, select a week, filter out elective modules you don't share, and APTime will highlight all the common free slots on a visual timetable grid.

**Features:**
- Compare timetables across multiple APU intakes
- Filter by group number (G1–G5)
- Select current or upcoming week
- Ignore elective CT modules not shared by everyone
- Visual timetable grid with free slot highlights
- Clean two-panel dashboard layout

Tech stack: Python · Flask · HTML · CSS · JavaScript · APSpace Open API
